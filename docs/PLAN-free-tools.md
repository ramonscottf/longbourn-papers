# Longbourn Free Tools — "The Writing Desk" (giveaway strategy)
Status: PLANNED · Owner: Scott · Builder: Skippy · 2026-07-03
Goal: things worth arriving for. Guides bring search traffic; tools earn shares/links;
the course captures email; email sells paper. Everything genuinely free, genuinely good.

## Hub: /writing-desk/
Free-resources index page in main nav: tools + guide collections + course signup.
One page, ships with Tool #1.

## Tool 1 — "What Do I Write?" (the wedge — build first)
Occasion → relationship → tone picker → structured guidance: 6–8 real opening lines,
body prompts, closings, what-to-avoid; copy buttons. Pure static JS on the worker
(no backend). Each occasion deep-links from its matching Pillar-A guide and back.
Email-gated extra: printable one-page cheat sheet PDF (capture via Resend audience —
EMAIL_MODE rules apply; wiring is dry-run until Scott's go). Est: 1 session.

## Tool 2 — Calligraphy Practice Sheet Maker
Type a phrase → pick style (faux-calligraphy skeleton / copperplate w/ 55° slant lines) →
canvas renders ghost text + x-height/slant guides → print-perfect @100% (print CSS,
letter+A4). Client-side only. THE shareable: Pinterest/teacher catnip, natural backlink
magnet, pairs with Pillar-C guides. Est: 1–2 sessions (font metrics = the fiddly part).

## Tool 3 — "The Lost Art" — free 5-part email course
Landing page + Resend drip: 1) Why write · 2) The five-sentence note · 3) Occasions
cheat-course · 4) Your hand is good enough (calligraphy starter) · 5) The stationery
wardrobe. Scott's videos slot in per lesson as they're made (course works text-first).
Capture = the list that eventually sells paper. Est: 1 session skeleton; content iterates.

## Sequence
1. Tool 1 + /writing-desk/ hub (ships with next content batch — guides cross-link day one)
2. Tool 3 skeleton (capture live early; list compounds)
3. Tool 2 (the linkbait, worth the extra session)
Growth loop: guide ranks → tool linked → sheet/course captured → drip → shop.
Rule: no dark patterns — gate only the PDF/course, never the tools themselves.
